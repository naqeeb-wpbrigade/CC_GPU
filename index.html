GPT-OSS 20B · MoE (21B total / 3.6B active per token)
MXFP4 weight ~16 GB · 16 GB VRAM minimum · 24 GB recommended · special vLLM build required
Special vLLM build required: Standard pip install vllm will not work. Use: uv pip install --pre vllm==0.10.1+gptoss --extra-index-url https://wheels.vllm.ai/gpt-oss/. Native MXFP4 requires Ada Lovelace (RTX 40-series) or Hopper (H100) — older GPUs use a Triton kernel fallback which works but runs slower.
